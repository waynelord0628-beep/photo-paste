# 酷酷的照片黏貼表

一個用 PyQt6 製作的圖片彙整工具，可將多張照片自動排版並輸出成 Word 文件（.docx）。

---

## 功能特色

- 多張照片一次選取，自動依檔名排序
- 六種排版模式，可隨時切換：
  - 1 頁兩張左右（含檔名 / 純編號）
  - 1 頁兩張上下（含檔名 / 純編號）
  - 1 頁三張橫式（含檔名 / 純編號）
- 右側即時預覽縮圖，可拖曳調整照片順序（iOS 風格）
- 雙擊卡片名稱可就地編輯單張照片名稱
- 批量改名工具列：指定範圍 + 新名稱前綴，自動加流水號
- 單張點 ✕ 刪除照片
- 選圖後自動帶入預設輸出資料夾
- 頁首標題可自訂
- 執行時背景處理，UI 不凍結

---

## 環境需求

- Python 3.10+
- 相依套件：

```
pip install PyQt6 python-docx Pillow
```

---

## 執行方式

```bash
python py檔/ui.py
```

或直接執行打包好的 exe：

```
dist/酷酷的照片黏貼表.exe
```

## 專案結構

```
.
├── README.md
├── word模板別動.docx          # 直向 Word 模板

│   └── 酷酷的照片黏貼表.exe   # 打包好的執行檔
└── py檔/
    ├── ui.py                  # 主程式（PyQt6 UI + 排版邏輯）
    ├── common.py              # 共用工具函式
    └── archive/               # 備份用獨立腳本
```
